## 特点

1. 完全的去中心化
2. 简洁而易于上手的UI
3. 轻量级的软件设计——但功能齐全！

## SWDChat 2.1.0的新特性
1. 新增托盘区图标
2. 新增文件分享与下载页面
3. 图片使用预览框，使之大小合适，单击打开
4. 启动动画
5. 应用新的按钮图标
