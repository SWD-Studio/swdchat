## 运行源码版本
配置环境并运行
##### Windows Powershell
```
pip install -r requirements.txt
py swdchat.py
```

## 系统要求：
操作系统：Windows 7 SP1 或更高

Python: CPython >=3.8 (不支持自由线程的Python构建)

硬盘：50MB及以上空闲空间

## 开发环境：
系统：Windows 11 25H2

Python解释器：CPython 3.13.11 for Windows(x64)
